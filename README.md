# advancing-autism-diagnosis
This is where I'll showcase my project about using Machine Learning for Autism Diagnosis.

I started this project as a junior in highschool (2023) and would like to continue to improve it over time. During the inital development period, I worked on three things: 
- Training the actual ML model itself (~95% accurate)
- Making it accessible to the public through a web page
- Writing a paper about the entire process

I'm sure there have been many advancements in our understanding of Machine Learning and the models themselves since I last worked on this project. However, I'll be including what I currently have in here on my github profile.
